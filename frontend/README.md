## ğŸ” Authentication Flow

1. User clicks "Connect MetaMask"
2. MetaMask prompts for connection approval
3. Frontend initiates Cognito custom auth flow
4. Cognito Lambda generates a nonce
5. User signs the nonce with MetaMask
6. Cognito Lambda verifies the signature
7. User receives JWT tokens (ID, Access, Refresh)
8. Frontend stores tokens and redirects to dashboard

## ğŸ¨ Components

## ğŸ“ SAA-C03 Concepts

### Performance

- **S3 Static Hosting**: Serverless, scalable
